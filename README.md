📌 Banking Management System

A web-based Banking Management System built using Spring Boot and Maven.
This project allows users to manage bank accounts and perform basic banking operations.

🚀 Features

Create new bank account

Deposit money

Withdraw money

View account details

Basic validation and error handling

🛠️ Tech Stack

Java

Spring Boot

Maven

JSP / Servlet (if used)

MySQL (if connected)
